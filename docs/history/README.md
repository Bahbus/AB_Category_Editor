# Durable History Index

This directory stores detailed, repository-versioned review and phase evidence
that is useful for targeted research but unnecessary for ordinary startup
context.

## Archives

- [`PHASES_27_77.md`](PHASES_27_77.md) — the complete detailed review record
  from the earlier project foundations through Phase 77, moved from the former
  primary `REVIEW_HISTORY.md` during Phase 78.

## Archive policy

- Every archive must be linked here and from `../REVIEW_HISTORY.md`.
- Historical sections have one detailed home; do not copy the same phase
  narrative into multiple archives or primary documents.
- Preserve exact evidence and decisions when archiving. A concise current
  summary may link here, but Git history alone is not a substitute for the
  repository archive.
- Add a new archive only when detailed evidence would materially enlarge a
  primary entry point. Use descriptive ranges and keep chronology discoverable.
